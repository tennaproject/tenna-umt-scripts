# Tenna UndertaleModTool Scripts

A collection of [UndertaleModTool](https://github.com/UnderminersTeam/UndertaleModTool) scripts for Deltarune created by Tenna Project.

## Installation

1. Download or clone this repository
2. Copy the `tenna-umt-scripts` folder to your UndertaleModTool `Scripts` folder
3. Open UndertaleModTool

## Usage

1. Open UndertaleModTool and load your `data.win`
2. Run `UTDR Scripts/Debug.csx` first to enable debug mode (it's required for other scripts to work)
3. Run any other scripts you need from `tenna-umt-scripts/`
4. Save your modified `data.win`
5. Launch the game

> **Note:** To remove modifications, reload a fresh `data.win` from your game backup.

## Scripts

### `FlagWatcher.csx`
![flagwatcher.png](/assets/flagwatcher.png)

Shows real-time flag changes on screen with timestamps.

- Displays the last 20 flag changes in the top-right corner
- Timestamps in `M:SS` format
- Yellow text with shadows for high visibility
- *Ignores* flag 33 changes
